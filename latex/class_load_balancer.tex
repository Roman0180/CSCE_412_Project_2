Sends a request to an available web server and manages the reply. 

This function iterates through the web servers in the web\+Servers vector. If a web server is available (i.\+e. has no active requests), it sends a request to it by popping it from the queue and setting the active\+Requests and curr\+Request fields of the web server. It also sets the finished\+Time field of the web server to the current clock cycle plus the time it takes to process the request.
